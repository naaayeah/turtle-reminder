# 🐢 거북이 알림

장시간 컴퓨터 작업 시 자세 교정을 도와주는 앱입니다.
설정한 시간 간격마다 알림을 보내줍니다.

macOS / Windows 모두 지원합니다.

---

## 시작하기

### 1. Node.js 설치

👉 [nodejs.org](https://nodejs.org) 에서 LTS 버전 다운로드 후 설치

### 2. 소스코드 다운로드

우측 상단 **Code → Download ZIP** 클릭 후 압축 해제

---

## 실행 방법

### macOS

1. 터미널 앱 열기
2. 아래 명령어 실행:
```bash
cd ~/Downloads/turtle-reminder-main
npm install
npm start
```

3. 상단 메뉴바에서 거북이 아이콘 클릭

---

### Windows

1. 시작 메뉴에서 **명령 프롬프트** 또는 **PowerShell** 열기
2. 아래 명령어 실행:
```bash
cd %USERPROFILE%\Downloads\turtle-reminder-main
npm install
npm start
```

3. 우측 하단 시스템 트레이에서 거북이 아이콘 더블클릭

---

## 사용 방법

1. 거북이 아이콘 클릭
2. 알림 간격 설정 (15분 / 30분 / 1시간 또는 직접 조정)
3. **시작** 버튼 클릭
4. 설정한 간격마다 알림 배너 표시

---

## 기능

- 메뉴바 / 시스템 트레이 상주
- 빠른 간격 선택 (15분 / 30분 / 1시간)
- 시간 / 분 세부 조정
- 알림 사운드 ON/OFF
- 다크모드 자동 지원
- 거북이 트랙 타이머 시각화
